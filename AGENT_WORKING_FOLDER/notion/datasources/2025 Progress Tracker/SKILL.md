description: Skill for adding items to the 2025 Progress Tracker database.
---

Creation Rules:
1. **Always ask for the task category** (required).
2. **Status** defaults to `Not started` if the user does not specify one.
3. **Date** defaults to **today's date in ISO format** (`YYYY-MM-DD`) if the user does not specify one.
